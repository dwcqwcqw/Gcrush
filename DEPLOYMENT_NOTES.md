### 🔑 API密钥配置

本地部署需要创建 `config.local.js` 文件：

```javascript
const LOCAL_CONFIG = {
    RUNPOD_API_KEY: 'YOUR_ACTUAL_RUNPOD_API_KEY_HERE'
};

if (typeof CONFIG !== 'undefined') {
    Object.assign(CONFIG, LOCAL_CONFIG);
}
```
